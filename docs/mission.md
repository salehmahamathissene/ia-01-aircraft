# IA-01

Inspect Africa Aircraft 01

## Vision

Design the first aircraft developed under Inspect Africa Aerospace.

The aircraft will be optimized for African operations including:

- Remote airstrips
- Humanitarian missions
- Pilot training
- Inspection missions
- Light cargo
- Surveillance
